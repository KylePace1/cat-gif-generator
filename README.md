# Cat Clicker Game

An addictive clicker game where you collect cat points, unlock upgrades, and earn achievements! Features random cat GIFs, lofi background music, and explosive animations.

## Features
- **Clicker Game Mechanics**: Click to earn cat points and level up
- **Upgrades System**: Unlock 4 different upgrades to increase your click power
- **Achievements**: Earn 6+ achievements as you progress (Cat Novice, Cat Master, Cat Legend, Cat Emperor, Cat God!)
- **Lofi Music Player**: Toggle background music for a chill gaming experience
- **Explosive Animations**: Every 5th click triggers a spectacular cat explosion with +50 bonus points!
- **Progress Saving**: Your game progress is automatically saved to local storage
- **Beautiful UI**: Modern gradient design with smooth animations
- **Responsive Design**: Works on mobile and desktop

## Live Demo
Visit the live site: [Cat Clicker Game](https://kylepace1.github.io/cat-gif-generator/)

## How to Play
1. Click the "Generate Cat" button to earn points
2. Collect cat points to level up
3. Purchase upgrades to increase your click power
4. Unlock achievements as you progress
5. Watch out for the cat explosion every 5th click!
