# Cat GIF Generator

A simple and fun web app that generates random cat GIFs using the Giphy API.

## Features
- Random cat GIF generation
- Clean, modern interface
- Responsive design
- Smooth animations

## Live Demo
Visit the live site: [Cat GIF Generator](https://yourusername.github.io/cat-gif-generator/)

## How to Use
Simply click the "Generate Cat GIF" button to see a random cat GIF!

## Technologies Used
- HTML5
- CSS3
- JavaScript (Fetch API)
- Giphy API
